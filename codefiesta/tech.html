TECH WIZARD



OBJECTIVE

This event takes the participant on a journey testing his skills on all broad domains of technology. From general knowledge about language and tech to coding a program in the basic languages to testing the hardware skills, in a broad perspective of course, like in assembling the CPU.

ROUND 1

All MCQ's will be on programming languages.Time limit is 15 minutes.

ROUND 2

This will be the coding round,it can be done in any language.Test cases will be given through which the participant has to code a program according to the problem statement. The problem statement will be hidden in a directory, whose location has to be found by the participant from a given clue. Fastest and efficient code qualifies. Time limit is 30 minutes.

ROUND 3

This is entirely hardware based in which the mother board has to be assembled. The fastest and the acurately assembled CPU wins. Time limit is 30 minutes.
